VS Code addon module for [Lua Language Server](https://marketplace.visualstudio.com/items?itemName=sumneko.lua) extension with complete Luanti (Minetest) API annotations.

## Installation

You can find and install **"Luanti Full API"** addon via [LLS Addon Manager](https://luals.github.io/wiki/addons/#addon-manager).

[Other ways to install "Luanti IDE Helper"](https://github.com/Voxrame/luanti-ide-helper?tab=readme-ov-file#supported-ides--plugins) (separate extension, Jetbrains, othe IDEs & editors)

## Features

 - Complete Luanti API annotations
 - Type definitions and auto-completion
 - Documentation hints for all functions and types of Luanti

## Related
 
All annotations are in: [luanti-ide-helper](https://github.com/Voxrame/luanti-ide-helper)
